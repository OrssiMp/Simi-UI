Documentation Simi UI

Guide complet d'utilisation de la bibliothèque Simi UI

Introduction

Simi UI est une bibliothèque de composants UI moderne construite sur Bootstrap 5.3.7 et Tailwind CSS 3.4.17. Elle offre une collection complète de composants réutilisables, personnalisables et accessibles pour créer des interfaces utilisateur professionnelles.

Installation

1. Inclure les fichiers CSS

Ajoutez les liens suivants dans la section <head> de votre HTML :

<!-- CSS de base (Bootstrap + Tailwind) --> <link rel="stylesheet" href="similar.css"> <!-- Composants Simi UI --> <link rel="stylesheet" href="styles/simi-original/simi-components.css"> <!-- Thèmes --> <link rel="stylesheet" href="styles/themes.css">

2. Inclure les fichiers JavaScript

Ajoutez le script avant la fermeture de la balise </body> :

<!-- Version complète --> <script src="js/similar.bundle.js"></script> <!-- OU version minifiée (recommandé en production) --> <script src="js/similar.bundle.min.js"></script>
Note : Le fichier similar.css contient à la fois Bootstrap et Tailwind CSS. Vous pouvez utiliser les classes des deux frameworks simultanément.

Structure des Fichiers

smi_ui/ ├── index.html # Page d'accueil ├── similar.css # CSS de base (Bootstrap + Tailwind) ├── styles/ │ ├── themes.css # Système de thèmes │ ├── simi-original/ # Composants basés sur similar.css │ │ └── simi-components.css │ └── simi-custom/ # Composants personnalisés │ └── simi-custom-components.css ├── js/ │ ├── similar.bundle.js # JavaScript complet │ └── similar.bundle.min.js # Version minifiée ├── sass/ # Sources SASS │ ├── main.scss │ ├── abstracts/ │ ├── components/ │ ├── layout/ │ └── utilities/ ├── pages/ # Pages de documentation │ ├── components/ │ ├── layout/ │ └── utilities/ └── logo/ # Logos du projet

Utilisation des Composants

Boutons

Utilisez les classes simi-btn pour créer des boutons :

<button class="simi-btn simi-btn-primary">Bouton Primary</button> <button class="simi-btn simi-btn-secondary">Bouton Secondary</button> <button class="simi-btn simi-btn-success">Bouton Success</button>

Cartes

<div class="simi-card"> <div class="simi-card-body"> <h3>Titre de la carte</h3> <p>Contenu de la carte</p> </div> </div>

Formulaires

<div class="simi-form-group"> <label class="simi-form-label">Email</label> <input type="email" class="simi-form-input" placeholder="email@example.com"> </div>

Système de Thèmes

Simi UI inclut 5 thèmes prédéfinis :

  • Light - Thème clair (par défaut)
  • Dark - Thème sombre
  • Blue - Thème bleu
  • Purple - Thème violet
  • Green - Thème vert

Changer de thème

// En JavaScript document.documentElement.setAttribute('data-theme', 'dark'); // Sauvegarder dans localStorage localStorage.setItem('simiui-theme', 'dark');

JavaScript API

Modales

// Ouvrir une modale SimiUI.openModal('myModal'); // Fermer une modale SimiUI.closeModal('myModal');

Toasts (Notifications)

// Afficher un toast SimiUI.showToast('Message de succès!', 'success', 3000); // Types disponibles: 'success', 'error', 'warning', 'info'

Alertes

// Créer une alerte dynamique SimiUI.createAlert('Message important', 'warning', 5000);

Copier dans le presse-papiers

SimiUI.copyToClipboard('Texte à copier');

Utilisation de SASS

Pour personnaliser les styles, modifiez les fichiers SASS dans le dossier sass/ :

Variables personnalisées

Éditez sass/abstracts/_variables.scss :

// Couleurs personnalisées $primary: #0d6efd; $secondary: #6c757d; // Espacements $spacing-base: 1rem; // Bordures $border-radius: 0.5rem;

Compiler SASS

# Installer SASS npm install -g sass # Compiler sass sass/main.scss styles/compiled.css # Watch mode (recompilation automatique) sass --watch sass/main.scss:styles/compiled.css

Classes Utilitaires

Espacement

Classe Description
m-{0-10} Margin sur tous les côtés
mt-{0-10} Margin top
p-{0-10} Padding sur tous les côtés
pt-{0-10} Padding top

Affichage

Classe Description
d-none Masquer l'élément
d-block Display block
d-flex Display flex
d-grid Display grid

Composants Spéciaux (-s-)

Les composants avec le préfixe -s- sont des composants avancés avec des fonctionnalités spéciales. Consultez la page dédiée pour plus d'informations.

Important : Les composants -s- peuvent nécessiter JavaScript pour fonctionner correctement. Assurez-vous d'inclure similar.bundle.js.
