═══════════════════════════════════════════════════════════════════════════════
  UTILISATION
═══════════════════════════════════════════════════════════════════════════════

1. Ouvrir index.html dans un navigateur
2. Explorer les composants via la sidebar
3. Utiliser la recherche pour trouver rapidement
4. Changer de thème avec le sélecteur
5. Copier les codes d'exemple
6. Consulter la documentation complète

Pour compiler SASS:
  sass sass/main.scss styles/compiled.css
  sass --watch sass/main.scss:styles/compiled.css
