  COMMENT UTILISER
═══════════════════════════════════════════════════════════════════════════════

1. OUVRIR LE SITE:
   • Double-cliquez sur index.html
   • Ou ouvrez avec un serveur local

2. TESTER LES THÈMES:
   • Cliquez sur "🎨 Thèmes" dans le header
   • Sélectionnez un thème prédéfini
   • Ou utilisez le color picker pour un thème personnalisé

3. TESTER LE MENU MOBILE:
   • Réduisez la fenêtre < 768px
   • Cliquez sur le menu hamburger (☰)
   • La sidebar s'ouvre depuis la gauche

4. EXPLORER LES COMPOSANTS:
   • Cliquez sur "Full & Less" dans la sidebar
   • Voir les exemples de composants Stateful/Stateless
   • Copier le code HTML

5. UTILISER LES ICÔNES:
   • Incluez icons/simi-icons.css
   • Utilisez <i class="si si-home"></i>
   • Ajoutez des tailles: si-lg, si-2x, etc.

6. SIGNALER UN BUG:
   • Cliquez sur "Signaler un Bug" dans la sidebar
   • Configurez EmailJS (voir emailjs-config.txt)
   • Remplissez le formulaire

7. VOIR TOUTES LES COULEURS:
   • Allez dans Utilitaires > Couleurs
   • Cliquez sur une couleur pour copier le code
